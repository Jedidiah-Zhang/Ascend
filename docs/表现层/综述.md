# 表现层 — 综述

表现层定义玩家感知到的一切：视觉、界面、音频。不包含游戏逻辑，只负责"呈现"。

## 子模块关联

```
视觉风格 → 角色外观
    ↓         ↓
  界面交互 ← 音频音效
```

- **视觉风格**定义场景、tile、美术方向
- **角色外观**定义基因如何驱动角色视觉差异（部位拼接、颜色、纹理）
- **界面交互**定义 HUD、面板、菜单、操作反馈
- **音频音效**定义环境音、动作音、对话音

## 与其他大类的接口

- ← **世界框架**：地图、群系、季节的视觉呈现
- ← **基因系统**：基因效果在角色外观上的可视化
- ← **生命个体**：受伤、疾病、需求状态的外观表现
- ← **群体社会**：关系值、记录、管理界面的 UI 呈现
- ← **玩家行动**：所有玩家操作的界面和操作反馈
