# 视觉风格 — 设计

## 场景表现

- 2D 俯视 / 等轴侧（isometric）简约风，像素画风（纹理最近邻过滤，无平滑）
- 模块化 tile 构成建筑、自然场景，spritesheet 纹理 + TileSet 复用
- 地图类似 Minecraft，有不同地形，自动生成，无限延伸
- **地形有海拔起伏**：详细地图按逐格海拔分层渲染，高处 tile 在屏幕上整体上移，形成台地/阶梯式立体感（每层 16px 视觉偏移）
- 小人类似 jrpg，使用不同图层拼接出整体外观（未实现：当前为单张占位 Sprite）
- 基因导致的外观差异以简单形状或纹理呈现（未实现）

## 地图呈现

- 玩家始终位于视野中心，相机跟随玩家；鼠标滚轮缩放（0.15x–4x）
- 地图以 chunk（200×200 tile）为单位流式加载：走到哪加载到哪，远处区块卸载，玩家无感知
- 地形类型（9 种：深水/浅水/沙滩/平原/丘陵/山地/雪峰等）映射到 spritesheet 中的固定 tile 图案
- 水面（河流/湖泊/海洋）固定在海拔 0 层，陆地按后端下发的逐格海拔数据分层

## 配色基调

- 全局背景深灰蓝（#1A1A1F），突出地图主体
- 调试 UI 统一半透明黑底 + 绿色标题 + 浅灰正文，不遮挡游戏画面
