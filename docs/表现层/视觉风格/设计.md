# 视觉风格 — 设计

## 场景表现

- **固定视角 3D**：正交投影相机，俯角锁死 60°，正北朝向。可读性优先——始终保持等轴策略视角的全局掌控感
- **低模像素风**：3D 模型用低多边形（low-poly）风格，纹理用像素点阵贴图（最近邻过滤），保持原有的像素美学
- **混合管线**：核心场景 3D 渲染 + 2D CanvasLayer 叠加 UI（终端、F3 面板、事件日志等保持不变）
- **地形真实海拔**：GridMap 程序化生成地形网格，每个 tile 的顶点高度直接来自后端海拔数据（米），立体感自然真实——不再需要伪 3D 的 Y 偏移模拟
- **角色与生物**：Sprite3D + billboard 模式（始终面朝相机），2D spritesheet 动画帧驱动。固定视角 + 正交投影下无透视畸变
- **植物与灌木**：Sprite3D cross-plane（X 形交叉双面贴图），经典俯视策略游戏做法
- **建筑与岩石**：3D 低模 mesh（AI 生成 + 手动调校），带 PBR 材质，数量少品种少
- **基因外观差异**：角色 Sprite3D 贴图可做分层叠加（基础体 + 基因变异层），简单纹理/颜色叠加即实现

## 地图呈现

- **固定正交相机**：俯角 60° 正北，纯正交投影，无近大远小。玩家始终画面中心，鼠标滚轮缩放
- **GridMap 流式加载**：地形以 chunk（200×200 cell）为单位，每 chunk 一个 GridMap 节点。加载/卸载策略沿用现有流式逻辑（LRU + 视野半径）
- **地形类型**：9 种（深水/浅水/沙滩/平原/丘陵/山地/雪峰等），映射为 GridMap MeshLibrary 中的不同 mesh 项。水面层单独一个 GridMap 或 shader 材质（半透明 + 动画水位）
- **环境表现**：WorldEnvironment 提供全局后处理——大气散射、雾效、色调映射，配合天气系统实时调整（晴天/阴天/暴风）

## 光照方案

- **DirectionalLight3D** 模拟太阳：方向由游戏时间驱动（昼夜循环），投射实时阴影
- **环境光** 保证暗面可见，强度由天气/日照参数调节
- **点光源/聚光灯** 用于夜间火把、营火、建筑窗户等局部光源
- **遮挡虚化**：相机到玩家位置 raycast，命中树冠/屋顶 → Tween alpha 到 0.2；不再遮挡 → 恢复 1.0。用 dithering 溶解而非生硬变透明

## 洞穴与地下

- 洞穴层为独立 Node3D 子场景，地表网格在入口处 Area3D 触发后隐藏屋顶
- 洞穴内可加局部点光源（火把），与地表光照隔离
- 地下区域靠独立的 GridMap + 暗色调环境，深度缓冲自动处理遮挡

## 天气视觉

- **GPUParticles3D**：雨滴、雪花、沙尘、落叶——3D 粒子系统替代 2D 粒子，天然有深度和密度感
- **雾效**：WorldEnvironment 的 volumetric fog 由后端湿度/温度参数驱动浓度和颜色
- **云层**：简单的半透明移动 plane 或 particle 云团

## 配色基调

- 全局背景色由 WorldEnvironment 的 clear color 控制，深灰蓝基调
- 调试 UI 不变——CanvasLayer 覆盖在 3D 视口之上，半透明黑底 + 绿色标题
- 白天/黄昏/夜晚通过 DirectionalLight3D 颜色 + 环境光强度 + 后处理色调映射实现

## 相机约束

| 自由度 | 状态 |
|--------|------|
| 俯仰角 | 锁死 60° |
| 水平旋转 | 锁死正北 |
| 投影模式 | 锁死正交 |
| 缩放 | 鼠标滚轮自由 |

> 未来可考虑解禁为 45°-75° 俯仰 + 90° 吸附旋转（Q/E 键），但初期锁死以降低工程复杂度。
