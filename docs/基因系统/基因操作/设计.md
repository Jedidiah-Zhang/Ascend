# 基因操作

- 基因模块是离散的，非连续数值
- 不存在最优基因组合
- 存在基因突变，退化的可能。需注意保持玩家在"设计生物"，而非"赌概率"
- 遗传系统的节奏需要把控。

## 基因操作的四种方式

| | 实验室编辑 | 育种 | 克隆 | 从零创造 |
|---|---|---|---|---|
| 做什么 | 将已知基因模块拼接到已有个体 | 两个体的基因组合产生后代 | 复制一个体的完整基因模板 | 用基因模块库从零合成新生命 |
| 速度 | 立即 | 一代时间 | 需培养时间 | 需大量培养时间 |
| 可控性 | 高 | 低（不可控遗传） | 完全复制 | **完全定制** |
| 输入 | 已有个体 + 基因模块 | 两个个体 | 一个体的完整基因 | 多来源基因模块 |
| **独特价值** | 精准优化 | 发现新基因组合 | 意识转移兼容性最高 | 不受受体限制的完全设计自由 |

- 育种是**基因发现的引擎**——产出自然界和实验室都不存在的新基因变体
- 实验室编辑是**基因应用的工具**——把已知基因拼接至目标个体
- 克隆是**意识转移的基础**——完美兼容的驱壳，但依赖已有模板
- 从零创造是**编辑的终极形态**——不需要受体，用已知基因模块合成全新个体。代价最高（需要顶级设施、大量资源和基因模块库），但给了你绝对的设计自由

从零创造产出的个体是**空白的**——没有个人经历、没有与他人的关系、没有技能沉淀。需要时间在群体中建立关系。与自然出生个体（出生即有关系网络）形成有意义的对比：基因完美，但社会孤立。

## 基因自动生成机制

开发者只需设计 10-15 个基础基因模板（标签 + 效果方向 + 数值范围），育种系统自动从模板参数空间中生成新基因变体。

### 重组生成（育种产出）

```
父基因A: [tag: metabolism] 采集效率 +30%, 食物消耗 +15%
母基因B: [tag: metabolism] 采集效率 +10%, 食物消耗 +5%

子代新基因C:
  采集效率 = random_range(10%, 30%)  → 22%
  食物消耗 = random_range(5%, 15%)   → 11%
  tag = metabolism（同标签继承）
  名称由系统自动组合（"代谢优化变体-γ"）
```

### 突变生成（低概率触发）

在重组基础上，小概率触发标签变异或交叉效果：A的metabolism标签和某个隐性因子交叉，子代基因新增了 social 标签效果。

### 命名规则

系统从效果类型 + 变体后缀自动生成名称，无需人工命名每个新基因。

组合空间：10个基础模板 × 每种2-3个效果 × 连续数值区间 × 标签混合 = 数百种可生成的变体，远超手写规模。

## 基因改造交互设计

### 兼容性矩阵

每个基因模块携带标签（如 `metabolism`, `sensory`, `structural`, `social`）。在实验室界面中，向 NPC 添加基因时显示冲突/增益矩阵：

| | 基因A (代谢) | 基因B (结构) | 基因C (社交) |
|---|---|---|---|
| 基因A | - | ✓ 无冲突 | ✗ 排斥（同为代谢组，低优先级被抑制） |
| 基因B | ✓ | - | ◆ 协同："厚皮"+"伪装色"→解锁潜行加成 |

规则：同类相斥、互补增益、部分覆盖。矩阵让玩家看清选择空间，决策是"选 A 还是 B"而非"怎么摆"。

### 概率范围与定向修正

- 改造时显示效果的概率范围（"采集效率变化：+20% ~ +40%"），具体值在完成后确定
- 效果确定后不可重抽，但可以**追加投入**进行二次调整，成本递增
- 追加方向是确定的——玩家明确知道再投入会改善什么、改善多少
- 概率的价值体现在**不同存档之间基因组合不同**，而非同一局内反复抽取

### 实验室即时反馈

改造完成后，面板直接展示数值变化（"采集效率：1.0 → 1.3"）。能力、需求、群体偏好三层效果一目了然。冲突/抑制的基因用灰色标出。

后续行为记录的作用并非验证数值——数值已经明确——而是观察该 NPC 改造后在世界中与其他人产生的互动变化。

## 生物类型

生物分为两类，获取其基因的方式不同：

**非智慧生物**：没有灵魂。如动植物。可通过工具、陷阱等方法捕获活体，带回实验室研究。显性特征在野外可见（外观、器官），隐性特征在研究中揭示。

**智慧生物**：拥有与玩家同质的意识。包括玩家所在种族个体、其他智慧种群。也存在显性和隐性特征，但基因获取路径不同。

## 获取智慧生物基因的三种路径

### 路径 1：死后研究

族中有人死亡后，可以研究遗体提取基因。这是最自然的路径，但有限制：
- 死后基因开始降解，提取难度提升一级，部分稀有基因永不可得
- 不同死因影响可用基因（疾病致死可能污染样本）
- 必须在腐坏前完成研究（时间窗口）

这也引出一个潜在的伦理困境——如果存在极度渴望的稀有基因，玩家会等待其自然死亡，还是……？

### 路径 2：交换研究

不是说服对方"让我研究你"，而是你提供对方族群需要的东西以换取研究许可：
- 你治愈他们族群中的某疾病，他们允许你研究一个自愿者
- 你提供他们无法产出但急需的物资，交换一次研究机会
- 两个族群有共同威胁（环境、猛兽、敌对族群），你帮忙解决，获得研究权限

条件是**他们有自己的具体需求，而你恰好能提供解。**

### 路径 3：文化与信仰差异

不同智慧种族的价值观不同。极少数个体会因为自身文化的特殊性而自愿接受研究——但这不可预期，不是一种稳定策略。发现此类个体本身就是探索的稀有收获。

### 伦理光谱

这三种路径组合成一个有重量的决策范围：靠交换（慢、代价高）、靠等待（可能永远等不到）、靠偶然（不确定性）——或者在特殊情境下靠强制。玩家在沿着这个方向走多远，就是"意志能否保留"这一命题的体现。

## 基因提取难度与科技树

每个基因模块有提取难度等级。基础基因可在实验室初期直接提取，高难度基因需要科技树解锁。同一个体可能携带当前科技无法提取的基因——玩家可标记地点，科技达标后重返。

科技树推进条件：
- 已研究的新物种/变异个体数量
- 资源投入
- 已提取基因的累积数量

这形成了探索的长线动力：早期发现了某种稀有个体却无法提取其核心基因，留下"标记—回归"的悬念。
