# 基因系统 — 综述

基因系统是游戏的核心差异化机制。基因操作是玩家改变世界的主要手段。

## 子模块关联

```
基因操作 → 饮食兼容
    ↓
  个体系统（人格、生理、身体）
```

- **基因操作**定义四种改造方式（编辑、育种、克隆、从零创造）、兼容性矩阵、自动生成机制
- **饮食兼容**是基因系统的一个具体应用方向——消化基因决定食物吸收率，形成"改作物→改消化基因→大规模种植"的投产链

## 与其他大类的接口

- → **心智系统**：基因为 LLM 提供不可改变的性格底色
- → **生命个体**：基因同时影响生理需求参数、身体部位增减、人格分布参数
- → **群体社会**：基因影响关系记录的方式（同一事件不同基因记录不同看法）
- → **世界框架**：不同群系的生物携带不同基因，驱动探索
- → **玩家行动**：基因操作、育种、种植、畜牧都依赖基因系统
