# 饮食相容性

基因决定消化能力，作物决定消化需求。不是"偏好吃什么"，而是**能吃多少、吸收多少**。

## 消化系统

每种食物带一个消化需求标签，NPC 的基因组合决定消化能力标签。匹配度 = 营养吸收率：

```
食物 硬壳果 → [digest: fibrous_hard]
食物 荧光菇 → [digest: fungal_toxin]
食物 标准小麦 → 无特殊需求

NPC-A 有 [digest: fibrous_hard]   → 硬壳果 100% 吸收
NPC-A 没有 [digest: fungal_toxin] → 荧光菇 30% 吸收 + 可能中毒
NPC-B 两个都没有                  → 只能吃标准作物
```

## 消化基因赋予的能力

| 消化基因 | 能吃的食物类型 | 来源场景 |
|---|---|---|
| 粗纤维消化 | 硬壳果、树皮、根茎 | 野外稀有作物 |
| 真菌耐受 | 蘑菇、地衣类 | 洞穴生态产物 |
| 毒素代谢 | 含微量毒素的高产作物 | 高产作物的平衡代价 |
| 矿物吸收 | 含矿物沉积的作物 | 特定生物群落特产 |
| 广谱消化 | 所有普通食物 + 微量加成 | 代价：代谢更快，食量更大 |

## 两条交叉方向

**作物带相容性门槛**：你改造出一种高产小麦，但它携带了 `[digest: fibrous_hard]`。只有植入对应消化基因的 NPC 能充分利用。不改造人 → 只能种 70% 利用率的低效版。投产链：改作物 → 改消化基因 → 大规模种植。

**强基因需要特定食物维持**：一个强大基因（再生、夜视）带附加标签 `[requires: fungal_diet]`。不吃这些食物 → 基因效果打折或产生负面标签。强基因的自然制约——想要再生能力？先解决真菌种植问题。
