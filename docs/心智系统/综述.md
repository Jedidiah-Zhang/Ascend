# 心智系统 — 综述

心智系统是 NPC 的思想引擎。AI 是实现方式，心智是它代表的游戏概念。

## 子模块关联

```
AI原生NPC ←→ 标签与信息
     ↓
  技能列表 → 目标与愿望
     ↓          ↓
  所有其他大类（基因、个体、社会、玩家）
```

- **AI原生NPC**定义 LLM 驱动的决策架构、上下文组装方式、LLM 唤醒机制（显著性评分）
- **标签与信息**定义数值→标签→LLM 上下文的三层管道，是 AI 系统的数据基础设施
- **技能列表**定义 NPC 的能力边界——先天/习得/角色赋予三层来源，执行层通过技能模板的 preconditions 自动展开动作序列
- **目标与愿望**定义 NPC 的动机方向——当前目标 + 待办 + 长期愿望三层结构，按 priority + deadline 排序，目标完成后自动转化为持久角色身份
- 标签系统是 AI 感受世界的"感觉器官"——数值是原始信号，标签是经过过滤的信号，LLM 看到的是自然语言化的摘要
- 技能列表是 AI 的"能力边界"——需求产生目标后，技能过滤候选方案
- 目标与愿望是 AI 的"方向感"——LLM 调用之间保持行为连续性

## 与其他大类的接口

- → **基因系统**：人格维度作为 LLM 行为一致性的锚点；基因决定认知能力上限，影响技能习得数量
- → **生命个体**：生理需求和心理满足度输入动机层，驱动目标 priority 计算
- → **群体社会**：关系图谱为 LLM 提供社会上下文；角色身份的社会传播通过关系图谱同步
- → **玩家行动**：玩家搭话强制唤醒 LLM，绕过显著性评分
- → **世界框架**：物品和设施作为技能执行的资源支撑；环境变化触发目标条件丧失复查
