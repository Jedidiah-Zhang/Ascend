# 心智系统 — 综述

心智系统是 NPC 的思想引擎。AI 是实现方式，心智是它代表的游戏概念。

## 子模块关联

```
AI原生NPC ←→ 标签与信息
     ↓
  所有其他大类（基因、个体、社会、玩家）
```

- **AI原生NPC**定义 LLM 驱动的决策架构、上下文组装方式、性能策略
- **标签与信息**定义数值→标签→LLM 上下文的三层管道，是 AI 系统的数据基础设施
- 标签系统是 AI 感受世界的"感觉器官"——数值是原始信号，标签是经过过滤的信号，LLM 看到的是自然语言化的摘要

## 与其他大类的接口

- → **基因系统**：人格维度作为 LLM 行为一致性的锚点
- → **生命个体**：生理需求和心理满足度的当前值作为 LLM 动机层的输入排序
- → **群体社会**：关系图谱为 LLM 提供社会上下文（该 NPC 对对话对象的态度）
- → **玩家行动**：玩家与 NPC 对话触发 LLM 推理
