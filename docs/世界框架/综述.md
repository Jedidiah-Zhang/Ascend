# 世界框架 — 综述

世界框架定义游戏的物理基础：空间、时间、物质、事件。

## 子模块关联

```
地图生成 → 群系生态 → 季节气候
                ↑          ↓
                └── 时间系统 ──┘
                      ↓
              物品系统 ← 事件系统
```

- **地图生成**决定地形分布，地形决定群系分布
- **群系生态**定义每个区域的生物种类、资源分布、环境参数
- **季节气候**叠加在群系之上，产生时间维度的变化（温度、降雨、作物生长窗口）
- **时间系统**是季节和群系的驱动器——天的概念、年的概念
- **物品系统**定义世界中存在的实体物品属性——工具、食物、建材、基因样本等
- **事件系统**定义世界中的随机/脚本事件——天气突变、野兽袭击、全局状态触发

## 与其他大类的接口

- → **生命个体**：环境参数（温度、湿度）作为生理需求的输入；物品（食物）满足饱腹
- → **群体社会**：季节影响资源供给，间接影响社会稳定性
- → **基因系统**：不同群系的生物携带不同基因，驱动探索动机
- → **玩家行动**：种植采集依赖季节和群系；威胁事件触发防御需求
- → **表现层**：地图、群系、季节的视觉呈现
