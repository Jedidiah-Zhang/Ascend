# 物品系统 — 技术

## 数据结构

- Item: {id, type, count, weight_per_unit, volume_per_unit, durability?, durability_max?, tags[]}
- 工具类 count 固定为 1，不堆叠
- 生物样本: {id, source_creature_id, freshness, alive?}

## 容量

- 任何容器（人物/仓库/生物/载具）有 weight_capacity 和 volume_capacity
- 装入检查：sum(count × weight) ≤ weight_capacity AND sum(count × volume) ≤ volume_capacity
- 同 ID 物品不自动合并，由玩家手动管理

## 耐久

- 工具使用一次耐久 -1，耐久归零物品销毁
- 修理公式：材料数 = (max - current) / 修理倍率 × 修理物系数，可由基因或技能影响消耗比例数
- 修理不消耗最大耐久
