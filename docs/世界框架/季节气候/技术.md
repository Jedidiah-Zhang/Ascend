# 季节气候 — 技术

## 参数存储

- 每个参数的"年均基线"按分块存储，由噪声函数生成时确定
- 季节调制和每日波动实时计算，不持久化
- 参数值 = baseline + seasonal_offset(world_time) + daily_noise(world_time, seed)

## 噪声生成

- 分块的年均基线使用多层 Perlin/Simplex 噪声叠加，确保相邻分块渐变
- 不同参数使用不同频率和种子，避免温度和降雨量完全正相关

## 气候档位

- 气候类型是离散枚举值，分块生成时由年均温+年降雨量映射到对应的气候档位
- 每个气候档位定义各参数的基线范围和季节振幅范围

## 季节

- 季节函数嵌入气候类型，每个气候类型有独立的 seasonal_curve
- 温带：标准正弦，振幅由纬度决定
- 热带：双峰（旱季-雨季-旱季），振幅小
- 实现为查表 + 插值，避免运行时三角函数开销

## 基因适配判定

- 基因携带 condition 规则：{parameter, min, max}
- 天气参数超出规则范围时产生标签
- 判定频率：每日一次或状态变更时
