# 季节气候 — 设计

## 两层模型

```
气候（离散档位）              天气（连续值）
─────────────────           ─────────────────
群系分类的依据               此刻的真实数值
指导天气范围                  作为生理/种植/基因的输入
季节嵌入气候内部              随时钟实时波动
```

- **气候**：离散档位，描述一个区域的长期气象特征。用于群系分类、大致判断"这里冷不冷、干不干"
- **天气**：连续值，此刻的具体数值——温度是 23.7°C 而不是"温暖"。用于生理需求计算、作物生长判定、基因适应区间匹配

## 气象参数池

可扩展。初始六项：

| 参数 | 单位 | 说明 |
|---|---|---|
| 温度 | °C | 体感温度，影响生理需求（体温维持）、作物生长期 |
| 降雨量 | mm/天 | 影响土壤湿度、水体补充、洪涝事件 |
| 日照时长 | 小时/天 | 影响作物光合作用、某些生物的活动周期 |
| 海拔 | m | 决定气压（推导值），影响体能消耗和某些基因表达 |
| 湿度 | % | 相对湿度，影响腐烂速度、霉菌生长、体感舒适度 |
| 风速 | m/s | 影响体感温度、风暴事件、某些飞行生物的移动 |

海拔推导值：气压 = 1013.25 × (1 - 海拔/44330)^5.255，含氧量等比例变化。

参数池可扩展——后续发现需要新维度（如紫外线强度、土壤酸碱度）时直接加入。

## 三层叠加

每个分块的天气由三层叠加决定：

```
年均基线（噪声函数，分块生成时确定）
    +
季节调制（正弦波，由季节模块驱动）
    +
每日波动（小幅度随机，由天气系统驱动）
    =
此刻天气
```

- **年均基线**：分块生成时由噪声函数确定，是该分块该参数的长期均值。不同分块之间连续渐变
- **季节调制**：以年为周期的正弦波，振幅和相位由气候类型决定。温带气候冬夏温差大，热带气候温差小
- **每日波动**：小幅随机扰动，让每天不重样。幅度由气候类型和参数性质决定

## 气候档位

用于群系分类的离散标签。每个分块分配一个气候类型，参数的年均基线和季节振幅都落在该气候类型定义的范围内。

初始四种基础气候（后续可按需细分）：

| 气候 | 年均温 | 年降雨 | 季节特征 | 典型群系 |
|---|---|---|---|---|
| 热带 | 20-30°C | 高 | 温差小，分旱雨两季 | 雨林、稀树草原 |
| 温带 | 5-20°C | 中 | 四季分明 | 落叶林、草原 |
| 寒带 | -10-5°C | 低 | 冬长夏短 | 针叶林、苔原 |
| 干旱带 | 10-30°C | 极低 | 昼夜温差大 | 沙漠、灌木地 |

同一气候类型内部，不同分块因海拔、纬度差异，具体参数基线连续渐变。

## 季节

季节嵌入气候类型——不是独立的全局时钟切换。热带没有"春夏秋冬"，但有旱季雨季；温带有标准的四季；寒带只有短暂的生长季和漫长的冬天。

季节调制是气候参数上的连续正弦波——不是"入冬那天温度从 15 跳到 -5"，而是平滑过渡。

## 基因适配

每个天气参数定义生物的基因适应区间。例如：
- 耐寒基因：温度适应下限降到 -20°C
- 耐旱基因：年降雨量适应下限降到 200mm
- 喜光基因：日照时长适应下限提升到 10 小时

生物的实际适应区间由多个基因叠加决定。超出区间的个体产生负面标签（"寒冷不适""脱水"），通过标签系统送入 LLM 上下文。

## 与其他模块的接口

- → **群系生态**：气候档位决定群系分布；连续参数值决定群系内的具体物种构成
- → **生命个体**：天气参数直接输入生理需求计算（温度→体温、湿度→体感舒适、海拔→含氧量→体能消耗）
- → **基因系统**：天气参数区间作为基因适应性的判定基准
- → **玩家行动**：种植条件由天气参数的每日值决定（温度窗口、降雨量、日照时长）
- → **时间系统**：季节调制由世界时钟驱动；长跳期间气候参数以年均基线推进，跳过每日波动
