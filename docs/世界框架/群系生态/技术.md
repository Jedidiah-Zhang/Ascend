# 群系生态 — 技术

## 群系数据

- 每个群系类型一个数据模板，定义：地形参数范围、植物列表（权重）、生物列表（权重）、资源列表（稀有度+分布模式）
- 分块生成时由群系模板实例化，叠加噪声细节

## 微地形生成

- 群系模板定义子区域类型和出现概率
- 详细地图生成时，在群系模板约束下用次级噪声决定微地形
- 子区域边界自然渐变，不硬切

## 生物种群

- 每个分块维护该群系下的生物种群计数（不追踪每个个体的实时位置）
- 离屏时种群按生态模型自周转（捕食-繁殖-死亡）
- 玩家进入时种群实例化为具体生物个体，分布到子区域

## 群系扩展

- 新增群系只需要新增一个模板数据，不需要改代码
- 模板格式待 MVP 后定义具体 schema
