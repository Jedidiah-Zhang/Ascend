# 群系生态 — 设计

## 群系 = 气候 + 生态内容

群系是气候参数在地上的具象化。一个分块的气候档位确定后，群系档内细分（2 子型）由连续场决定。群系决定了这块地上有什么——地形、植物、生物、资源。

## 群系细分

每气候档细分 2 个子型，共 16 种陆地群系 + 3 种海洋群系。细分维度用连续场（降雨/温度/海拔/moisture 噪声），tile 生成时按隶属度加权混合 TerrainBias，保证 chunk 边界连续。

| 气候档 | 子型（低→高） | 细分维度 | 值域 |
|---|---|---|---|
| 热带雨林 | 热带季雨林 / 热带雨林 | 降雨 | 1500-2800mm |
| 热带草原 | 热带草原 / 热带疏林 | 降雨 | 600-1500mm |
| 沙漠 | 沙质沙漠 / 砾石戈壁 | moisture 噪声 | [-1,1] |
| 草原 | 矮草草原 / 高草草原 | 降雨 | 200-600mm |
| 温带森林 | 温带混交林 / 温带落叶林 | 温度 | 5-20°C |
| 亚寒带针叶林 | 北方湿地 / 北方针叶林 | 海拔 | 0-1500m |
| 极地苔原 | 极地荒原 / 苔原 | 温度 | -20~-5°C |
| 高山 | 高山草甸 / 高山裸岩 | 海拔 | 2000-3200m |

### 基础群系（初始实现）

#### 温带落叶林

- 气候：温带，四季分明
- 地形：缓坡为主，河流穿行，少量岩石露头
- 植被：阔叶乔木（橡、枫、桦），灌木丛，林间草地，秋季落叶覆盖地面
- 生物：鹿科、小型食草动物、掠食性动物（狼/熊）、鸟类、昆虫
- 资源：木材（硬木+软木）、浆果、药草、可采集真菌、浅层矿物
- 基因价值：温带适应性基因（四季耐受）、杂食消化基因、社交性基因（中等密度生物）

#### 沙质沙漠（原干旱灌木地）

- 气候：干旱带，昼夜温差大，年降雨极低
- 地形：流动沙丘，平坦或缓起伏，偶有干涸河床
- 植被：低矮耐旱灌木、多肉植物、稀疏草斑，绿洲聚集高价值物种
- 生物：爬行类、小型啮齿类、夜行性掠食者、毒虫
- 资源：沙、暴露地表矿物、特殊纤维（耐旱植物茎皮）、绿洲水源
- 基因价值：耐旱基因、节水代谢基因、高温耐受基因、夜行视觉基因

#### 砾石戈壁

- 气候：干旱带
- 地形：戈壁砾石地面，岩石露头多
- 植被：极稀疏耐旱灌木
- 生物：岩栖啮齿类、蛇、鹰
- 资源：暴露矿物、燧石、石料
- 基因价值：耐旱基因、岩栖适应基因

### 后续扩展方向

- 沼泽湿地：毒素抗性基因（可作为草原/热带雨林档的第三子型）
- 长生种聚落：人造群系类型，特殊 NPC 聚居

## 群系内结构

每个群系不是均质的——内部有微地形和子区域：

- **温带落叶林**中：林间空地（鹿群觅食）、河岸（水生资源）、密林深处（掠食者巢穴）、岩石区（矿物）
- **沙质沙漠**中：绿洲（高价值节点）、干涸河床（矿物+遗迹）、沙丘（危险生物）

微地形由详细地图生成时的噪声细节决定，群系只定义"可能有哪些子区域"。

## 跨群系边界

相邻分块群系不同时，详细地图上表现为渐变过渡带——几个 tile 宽的混合区（"温带落叶林边缘开始变干燥，灌木增多"），而非一刀切的边界线。

技术上，tile 生成时用 `biome_membership()` 算 tile 对各群系的隶属度（三角形隶属函数），TerrainBias 按隶属度加权混合。chunk 边界因连续场双线性插值 → 隶属度连续 → bias 连续 → 地形连续。

## 群系存在性

群系子型**不强制保证存在**（与气候档的强制注入不同）。理由：

- 群系是生态内容载体，注入到有生态意义的尺度会破坏自然分布
- 稀有群系是重玩价值的来源——"这个世界没有温带混交林"本身就是发现
- 软兜底：细分维度阈值落在大陆场实际值域内，保证统计上大概率出现

## 积雪

积雪线**不作为群系**——它的范围会随季节变化。当前 MOUNTAIN_PEAK 地形只由海拔带决定（临时占位），天气系统接入后改为动态雪线。

## 生物分布

群系决定生物**可能存在**的范围，并非每个分块一定刷满所有生物。实际分布由：
- 群系规定该物种的基础出现概率
- 地形子区域修正（鹿在河岸概率更高，熊在密林深处）
- 人口压力修正（聚落附近大型掠食者减少）
- 时间/季节修正（冬眠期熊不出现，迁徙物种随季节移动）

## 资源分布

同上——群系定义资源类型和基础丰度，子区域修正分布密度。关键资源（稀有药草、特殊矿物）在群系内不均匀分布，必须探索才能发现——这是驱动玩家"去新分块找东西"的基础动力。

## 与其他模块的接口

- ← **季节气候**：气候档位是群系的第一决定要素；天气连续值影响群系内生物活动和资源产出节奏
- ← **地图生成**：分块的群系分配由连续气候场 + moisture 噪声决定；群系通过 TerrainBias 反转来约束地图生成中的地形参数
- → **基因系统**：群系内的生物携带该群系对应的适应性基因，是玩家采集基因样本的天然来源
- → **玩家行动**：采集、狩猎、探索的内容由群系决定；群系难度梯度驱动玩家的远行方向
- → **时间系统**：长跳期间，群系内的生物种群按生态模型自我周转
