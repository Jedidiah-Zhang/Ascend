# 种植与采集

作物 = 非智慧生物。适用同一套基因系统，但基因复杂度低于动物（模板少，标签窄）。

## 作物类别

| 类别 | 主要产出 | 副产物 |
|---|---|---|
| 粮食作物 | 饱腹 | — |
| 纤维作物 | 纺织原料（衣物） | 油/饲料 |
| 药材 | 药物原料 | — |
| 饲料作物 | 畜牧饲料 | — |
| 功能作物 | 特殊原料（染料、建材） | — |

## 生长因子

| 因子 | 影响 |
|---|---|
| 土壤肥力 | 基础产量倍率 |
| 水分 | 灌溉/降雨，缺水→减产→枯萎 |
| 温度 | 季节联动，偏离适宜区间减产或死亡 |
| 光照 | 室外靠天，室内需设施补光 |

## 作物基因改造

| 改造方向 | 效果 | 代价 |
|---|---|---|
| 提高产量 | 每株产出↑ | 耗水↑、土壤肥力消耗↑ |
| 加快生长 | 生长周期↓ | 营养值↓ |
| 环境适应 | 扩大温度/水分适宜区间 | 产量略降 |
| 增加副产物 | 粮食作物同时产出纤维 | 主产物产量↓ |
| 抗病虫害 | 患病概率↓ | 生长周期略长 |
| 提升营养 | 食物饱腹恢复量↑ | 产量↓、生长更慢 |

## 采集 vs 种植

| | 采集 | 种植 |
|---|---|---|
| 规模 | 零散，靠野生再生 | 可大规模复制 |
| 效率 | 低 | 高 |
| 基建要求 | 无 | 耕地+灌溉 |
| 稳定性 | 受季节、野生动植物波动 | 设施加持下可控 |
| 品种 | 野外有什么采什么 | 种你筛选/改造过的 |

早期采集为主，中期过渡到种植，后期种植 + 基因改良产生巨大产出差异。采集不消失——野外永远有还没被发现的新品种。

## 季节

四个季节，影响采集、种植、畜牧、探索：

| 季节 | 种植 | 采集 | 探索影响 |
|---|---|---|---|
| 春 | 播种期 | 野菜、嫩芽 | 正常 |
| 夏 | 生长期 | 浆果、药材旺季 | 高温区需防中暑 |
| 秋 | 收获期 | 坚果、菌类 | 正常，迁徙动物活跃 |
| 冬 | 无法露天种植 | 极低，依赖储粮 | 低温区需防冻，稀有物种出现 |

- 不同生物群落季节节奏不同——热带冬夏差距小，温带四季分明，极地常年冬
- 基因改造可突破季节限制（耐寒作物可在冬季露天或温室生长）
- 冬季提供研究动力——种不了地时，做实验室研究、基因编辑、档案整理
