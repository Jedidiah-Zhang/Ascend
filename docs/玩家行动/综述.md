# 玩家行动 — 综述

玩家行动系统定义玩家能做什么。与前面的模拟层（世界、基因、个体、社会）不同，这里是交互层。

## 子模块关联

```
玩法进程（核心循环、节奏）
    ↓
┌───┼───┬───┬───┬───┬───┐
死  建  制  种  畜  经  威
亡  造  作  植  牧  济  胁
    设  工  采  驯  物  防
    施  具  集  化  流  御
```

- **玩法进程**定义核心循环和不同阶段的节奏变化
- **死亡机制**决定失败条件如何随进度演变
- **建造设施**和**制作工具**提供管理能力和物品消费出口
- **种植采集**和**畜牧驯化**是食物和基因来源的生产链
- **经济物流**解决群体规模上去后的资源分配
- **威胁防御**是基因探索延伸出的风险维度

## 与其他大类的接口

- ← 所有模拟大类为玩家行动提供操作对象和反馈
- → **心智系统**：玩家与 NPC 对话触发 LLM 推理
- → **表现层**：玩家的所有操作通过 UI 呈现；建造、种植的结果通过视觉反馈
