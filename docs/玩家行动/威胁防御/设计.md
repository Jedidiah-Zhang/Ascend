# 威胁与风险

威胁不是独立系统，而是基因探索的延伸：

- **环境威胁**：不同生物群系有生存门槛（寒冷、炎热、毒性）。环境不即死，但限制你在其中滞留的时长。给你改造自身以适应该环境的动机——比如捕捉耐寒生物，将其基因转入自身。
- **目标生物威胁**：携带稀有基因的生物往往也更危险。不是让你去打怪，而是制造决策——硬抓可能受伤和样本损坏；使用工具（陷阱/麻醉）需要资源和时间但安全；先标记再准备但下次它未必在。
- **自身基因的意外后果**：你改造的某个NPC因基因副作用而孤立、独处野外更久，它可能遇到危险——或发现稀有生物而你没有参与。这类威胁是你自己设计的。

捕获失败的结果通常是标本逃逸/损坏、自身受伤（暂时能力下降），仅在极端情况下致命。只有智慧生物的强制捕获存在关系崩塌风险。
