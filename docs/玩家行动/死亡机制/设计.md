# 失败与死亡

**核心定义：意识死亡 = 游戏失败。肉体死亡随进度逐渐不等于意识死亡。**

玩家作为世界的一员，设定上与 NPC 没有本质不同。但"死亡"的含义随游戏进程演变——不是一条直线从恐惧走向安全，而是每次降低肉体死亡惩罚时，都伴随着新的代价和阴影。

## 阶段 0：纯肉体（游戏开始）

- 肉体死亡 = 意识死亡 = Game Over
- 你没有后代、没有档案、没有传承机制
- 这个阶段的主题：**活下去**

## 阶段 1：档案记录

- 建造档案/记录设施后，知识不会随肉体死亡消失
- 新角色继承：已发现的基因配方、探索过的地图、建筑和物资
- 新角色丢失：与 NPC 的个人关系、未写入档案的短期记忆、上个角色的基因改造
- 关键限制：**需要主动去档案设施写入。** 若在野外死亡，上次写入之后获得的知识全部丢失
- 新角色是种群中另一个个体——可能是你的后代，也可能是任意同伴

## 阶段 2：意识载体

- 通过特定基因/技术改造，意识可以与肉体分离并转移到另一个身体
- 意识转移目标仅限于智慧生物——你转移进谁的身体？这是核心决策：
  - **自愿的同伴**：关系好、信任你、愿意为你承担风险。社会代价最高
  - **自己的后代**：基因可由你设计、身体更年轻。伦理上最复杂
  - **克隆体**：兼容性最高。需要极高科技线和大量资源
  - **创造个体**：完全由你设计的驱壳。兼容性不如克隆（无参照模版），但基因组合自由度最高。需要顶级设施和基因模块库
- 每次转移伴随代价：
  - 部分记忆永久消失（具体哪些无法完全预测）
  - 某些基因效果在新身体上表达不完全
  - 连续转移多次积累"意识退化"——当退化到极限，意识消散 = 真正的 Game Over
- 限制条件：
  - 需要专门的意识转移设施
  - 转移有冷却期，不能连续跳跃
- 新的决策：自然衰老然后转移（承受记忆损失），还是冒险继续活着寻找更好的转移时机？

## 阶段 3：意识独立

- 意识可短暂脱离肉体存在（存储于某种设施中）
- 此状态不可持续——意识逐渐衰减，最终消散，需要尽快找到新身体
- 此时解锁克隆技术：可事先制备与自身基因兼容的空白驱壳，替代"寻找宿主"的被动——但克隆需要大量资源，不是随手可得的
- 此时死亡的性质改变：不再怕被杀，而是怕"没有准备好的驱壳 + 存储设施断电"

## 死亡节奏设计要点

- 游戏允许频繁的小规模失败（基因退化、社会崩溃等），多为可修复、可调整、可逆的
- 真正的失败（意识死亡）通过小规模失败有较长的预兆期
- 死亡威胁由具体（狼、冬天、饥饿）逐渐转向抽象（意志消散、传承断裂）
- 玩家意外死亡应控制在游戏前期；后期死亡应是"知道自己要死了，最后的时间做什么"

_需要把肉体和精神分开看待。虽然不一定会做精神独立与肉体存在的情况，但是代码架构上最好分开。_
