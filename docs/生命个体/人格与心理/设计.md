# 人格维度

人格维度是基因赋予的稳定连续数值（0–1），描述 NPC 长期的行为倾向。

- **双极对称**：两极都是有效的生存策略，不存在"高=好、低=坏"
- **基因→分布、个体→抽样**：基因不直接设定人格值，而是设定正态分布的参数 (μ, σ)。每个个体从分布中随机抽取——同一对父母的后代人格相似但不相同
- **分布可遗传**：育种可定向塑造种群性格
- **人格是 LLM 的行为锚**：0.72 的社交性 = NPC 在 72% 的情境中选偏合群的回应，28% 可以偏孤僻——制造"像人"的不一致

| 维度 | 低值 (0.0) | 高值 (1.0) | 行为表现 |
|---|---|---|---|
| 社交性 | 孤僻 | 合群 | 低：独处舒适，社交消耗精力。高：社交充能，独处焦虑 |
| 勤勉性 | 懒散 | 勤勉 | 低：抵触持续劳动，偏好间歇。高：从完成工作中获得满足，闲置焦虑 |
| 秩序性 | 随性 | 有序 | 低：灵活适应，排斥僵化结构。高：按规则行事，混乱不安 |
| 支配性 | 顺从 | 支配 | 低：接受引导，被领导时不反抗。高：主动争取控制权，被命令时反弹 |
| 抚育性 | 疏离 | 抚育 | 低：回避照护责任，关系疏淡。高：主动关心弱者，从照护中获得满足 |
| 斗争性 | 协作 | 竞争 | 低：回避冲突，重视和谐。高：视冲突为常态，竞争中感到充实 |
| 刺激寻求 | 循规 | 求新 | 低：熟悉=安全，新事物=威胁。高：熟悉=厌倦，新事物=兴奋 |
| 时间视野 | 即时 | 长远 | 低：今天吃饱最重要。高：愿意为远期回报牺牲当下 |

**基因→人格的映射**：每个影响人格的基因带 `personality_shift: {dimension, μ_shift, σ_shift}`。多基因叠加构成个体的分布参数。σ 是该维度的不稳定度——σ 越大，个体在该维度表现越不可预测。

同一个事件，不同人格的 NPC 产生不同的行为记录和关系评价——这是人格在行为层面的直接体现。

# 心理满足度

六项心理满足度是**实时衰减的软约束**——不满足不会死，但产生行为惩罚（工作效率↓、拒绝社交、关系恶化等）。NPC 会自行尝试恢复，但人格维度决定了同一活动对不同 NPC 是充能还是耗能。

核心机制：**同一活动，不同人格的 NPC 获得相反的心理效果**。不是"高人格=恢复快"的量变，而是"一个充能、一个耗能"的质变。

## 社交满足

与他人互动的心理充实感。自然衰减：长期独处。

| 活动 | 高社交性（合群） | 低社交性（孤僻） |
|---|---|---|
| 群体交谈、协作 | **恢复** | **消耗** |
| 独处 | 消耗 | **恢复** |
| 被强制社交 | 恢复 | 大幅消耗 |
| 被强制隔离 | 大幅消耗 | 小幅消耗 |

## 自主满足

按自己意愿行事的感觉。自然衰减：被命令、被管控。

| 活动 | 高支配性 | 低支配性（顺从） |
|---|---|---|
| 做出影响群体的决策 | **恢复** | 消耗（决策压力） |
| 接受他人的命令 | 消耗 | **恢复** |
| 自由工作（无人干预） | 恢复 | 无变化 |
| 被强制劳动 | 大幅消耗 | 小幅消耗 |

## 能力满足

完成任务、克服挑战的成就感。自然衰减：长期无产出、反复失败。

| 活动 | 高勤勉性 | 低勤勉性（懒散） |
|---|---|---|
| 完成工作任务 | **恢复** | 小幅恢复 |
| 持续空闲/娱乐 | 消耗 | **恢复** |
| 被强迫工作 | 小幅恢复 | 消耗 |
| 任务失败 | 大幅消耗 | 小幅消耗 |

## 安全满足

对生存环境的安心感。一个数值，多事件入口——发生什么就走对应人格的调节，不存在子数值拆分。

| 安全来源 | 事件示例 | 调节人格 | 高值效果 | 低值效果 |
|---|---|---|---|---|
| 结构安全 | 执行固定日程、规则被遵守 | 秩序性 | **恢复** | 消耗（感觉束缚） |
| 资源安全 | 库存充足、作物长势好 | 时间视野 | **恢复** | 几乎无感 |
| 归属安全 | 被群体接纳、有人可靠 | 社交性 | **恢复** | 几乎无感 |
| 物理安全 | 野兽袭击、外敌入侵 | 无（不经过人格） | 全体扣减 | 全体扣减 |

LLM 动机层看到的不是拆分值，而是"安全满足 35/100，下降中，主要来源：资源焦虑"——最后那句由最近扣分最多的来源决定。

## 刺激满足

新鲜感、变化带来的心理激活。自然衰减：长期重复单调。

| 活动 | 高刺激寻求（求新） | 低刺激寻求（循规） |
|---|---|---|
| 探索新区域、新事物 | **恢复** | 消耗（不安） |
| 重复性劳动 | 消耗 | **恢复** |
| 突发事件/惊喜 | 恢复 | 消耗 |
| 长期无变化 | 大幅消耗 | 小幅消耗 |

## 亲密满足

深度一对一关系的情感联结。自然衰减：长期无深度互动。

| 活动 | 高抚育性 | 低抚育性（疏离） |
|---|---|---|
| 照顾/帮助特定对象 | **恢复** | 消耗（负担感） |
| 被他人照顾 | 小幅恢复 | **恢复** |
| 被期望承担责任 | 恢复 | 消耗 |
| 亲密关系破裂 | 大幅消耗 | 小幅消耗 |

性需求归属亲密满足——已有亲密关系对象的性行为作为特殊恢复源，大幅恢复亲密满足。软约束，不满足不会死亡，但长期不满足加速亲密满足衰减，可能产生"渴求"标签影响互动方式。

## LLM 上下文

每个满足度的当前值和变化方向输入动机层。LLM 看到的是自然语言化的状态——"我感到有些孤独，想找人聊聊"或"最近说话太多了，想一个人待着"。人格决定内容倾向，满足度数值决定紧迫程度。
