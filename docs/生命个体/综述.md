# 生命个体 — 综述

生命个体系统定义"一个活着的存在"从内到外的全部状态：生理、身体、心理。

## 子模块关联

```
生理需求 ←→ 身体部位 ←→ 人格与心理
    ↓          ↓           ↓
         基因系统（全部可被基因调节）
```

三者的互锁关系：
- **生理需求**是生存底线（饿、渴、累……不满足会死）
- **身体部位**是生理的物质载体（部位受伤→健康值下降→影响生理恢复）
- **人格与心理**是软约束层（不满足不会死但行为惩罚；人格决定什么活动回血、什么活动扣血）

## 与其他大类的接口

- ← **基因系统**：基因调节生理参数、增减身体部位、设定人格分布
- → **心智系统**：生理值和心理满足度输入 LLM 动机层作为行为优先级排序
- → **群体社会**：个体的需求状态影响其社会行为；关系好坏反过来影响心理满足度
- ← **世界框架**：环境温度、食物供给作为生理需求的外部输入
- → **玩家行动**：种植、畜牧、建造产出的物资满足生理需求；制作工具修补身体部位
