# 模块化身体部位

每个个体的身体由**基础部位 + 基因附加部位**动态拼接。部位是模块化的——基因决定了长什么样，受伤时部位独立追踪。

## 身体结构

```json
{
  "species": "human_base",
  "body_plan": {
    "head":       { "attachments": ["left_eye", "right_eye", "left_ear", "right_ear"] },
    "torso":      { "attachments": ["internal_organs"] },
    "left_arm":   { "attachments": ["left_hand"] },
    "right_arm":  { "attachments": ["right_hand"] },
    "left_leg":   { "attachments": ["left_foot"] },
    "right_leg":  { "attachments": ["right_foot"] }
  },
  "gene_additions": {
    "wings":  { "origin": "torso", "gene": "avian_wings_v3" },
    "tail":   { "origin": "torso", "gene": "prehensile_tail" }
  }
}
```

- 基础部位由物种模板定义，每个部位独立追踪健康值
- 基因附加部位（翅膀、鳃、角、触角等）由基因驱动增减
- 部位和功能挂钩：有翅膀才能飞，有鳃才能水下呼吸。部位受伤 → 功能降级或丧失

## 外伤

外伤具体到部位和类型，直接映射为 visible 标签：

| 外伤类型 | 表现 | 来源 |
|---|---|---|
| 割伤 | 部位健康↓，持续出血 | 利器、野兽爪击 |
| 瘀伤 | 部位健康↓，无出血 | 钝器、摔伤 |
| 烧伤 | 部位健康↓ | 火、高温 |
| 骨折 | 部位功能严重受限或丧失 | 重击、坠落 |
| 缺失 | 部位永久消失，功能归零 | 极端伤害、手术 |

## 内伤与疾病

不需要部位定位，用标签 + 严重度管理：

- **hidden 类**（需检查）：内出血、器官损伤、感染初期
- **visible 类**（外在症状）：发热、咳嗽、皮疹、呕吐
- **长期标签**（已确诊或持续）：慢性病、基因缺陷病、后遗症（"左腿旧伤，阴雨天疼"）

疾病有发展过程——感染从轻微到严重，治疗可截断。

## 基因与部位

| 基因方向 | 效果 |
|---|---|
| 厚皮/甲壳 | 割伤/瘀伤减免 |
| 骨骼强化 | 骨折阈值提高 |
| 再生能力 | 部位自愈速度↑ |
| 免疫强化 | 感染概率↓、疾病进展更慢 |
| 凝血增强 | 出血时间缩短 |
| 翅膀/鳃/角 | 新增功能部位 |

## 假肢

部位缺失可被制作系统填补——义眼、义肢。假肢可附带原装部位没有的功能（工具手、夜视义眼）。基因做不到的，科技来补。槽位始终保留，可替换、可升级。
