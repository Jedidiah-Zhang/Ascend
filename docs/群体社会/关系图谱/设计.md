# 关系图谱

**NPC 的身份不是预设标签，而是其他个体对它的主观看法的集合。**

每个 NPC 维护一个**关系页面**——记录该 NPC 对其他每个已知个体的看法：

- 关系值（正面/负面）
- 关系来源（共同采集、帮助过自己、冲突过等）
- 简短描述（如"可靠的采集者"、"上次偷我食物的家伙"——由经历自动生成）

同理，玩家对每个 NPC 也有自己的看法记录。

**NPC 的关系页面默认对玩家不可见。** 特定条件下可以获取：

- 与该 NPC 的信任度极高时
- 意识转移时，继承原角色的记忆片段
- 通过特定基因改造获得的洞察力
- 某种设施赋予的权限

这制造了有价值的**信息不对称**——你不能直接查询 NPC-A 对 NPC-B 的看法。你要么去问，要么获得特定能力/条件才能看到。

## 基因对关系的影响

社交偏好基因影响一个 NPC **如何记录和解读**他人的行为。同样是"分食物时少给了"，不同基因的 NPC 记录的看法可能完全不同——一个记录为"小事一桩"，另一个记录为"严重冒犯"。这意味着关系图谱不是客观中立的，而是被每个个体的基因"染色"过的。
