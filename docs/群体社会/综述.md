# 群体社会 — 综述

群体社会系统定义多体互动的规则和涌现：个体如何看待彼此、群体如何被模拟、管理如何产生。

## 子模块关联

```
关系图谱 → 世界模拟 → 管理制度
   ↑           ↓
   └── 互为基础 ──┘
```

三层递进：
- **关系图谱**是微观层——个体之间的主观看法的总和就是社会结构
- **世界模拟**是中观层——记录驱动的模拟使群体行为可追踪、可检索；视野内外切换保证性能
- **管理制度**是宏观层——规则从无到有、从口头到制度化的涌现过程

## 与其他大类的接口

- ← **基因系统**：基因影响关系记录方式（同一事件不同基因记录不同看法）
- → **心智系统**：关系图谱为 LLM 提供社会上下文；行为记录作为长期记忆素材
- ← **生命个体**：个体的心理满足度受社会关系影响（归属安全、社交满足）
- → **玩家行动**：管理能力由玩家建造的设施提供；NPC 管理者与玩家产生张力
- ← **世界框架**：季节和资源供给影响群体稳定性
